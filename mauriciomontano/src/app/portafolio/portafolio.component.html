Jul 2023 - Dic 2023

BREAST CANCER DETECTION ANALYSYS

This study presents a comparison of 3 machine learning algorithms: Multilayer Neural Network (MLP), Nearest Neighbor Search (KNN), and Support Vector Machines (SVM) on the Wisconsin Breast Cancer Diagnosis dataset (WDBC), measuring its accuracy in classification tests. The aforementioned data set consists of features that were calculated from digitized images of fine needle aspiration (FNA) tests on different breast masses. For the implementation of the machine learning algorithms, the data set was divided as follows: 80% for the training phase and 20% for the testing phase.

The dataset consists on ten characteristics recovered by image analysis: (1) radius, (2) texture, (3) perimeter, (4) area, (5) smoothness, (6) compactness, (7) concavity, (8) concave points, (9) symmetry and (10) fractal dimension. It also contains the mean value, extreme value (mean of the three largest values), and standard error of each feature, returning a vector of 30 real values.
